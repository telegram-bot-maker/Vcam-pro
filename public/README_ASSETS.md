# Vcam pro Website Static Assets Folder

Welcome! You can upload all your custom images, app screenshots, and icons directly into this `/public` folder using the AI Studio File Explorer.

Vite serves files inside the `public/` folder at the root path `/`.

## Recommended File Names to Upload:

1. **App Icon/Logo**:
   - Save your app logo as **`logo.png`** or **`logo.svg`** inside this folder.
   - You can then use it in code as: `<img src="/logo.png" alt="Vcam pro Logo" />`

2. **Screenshots**:
   - Save your Android screenshots as **`screenshot1.png`**, **`screenshot2.png`**, etc.
   - You can refer to them as `/screenshot1.png` in your source code.

3. **Backgrounds & Banners**:
   - Save your background covers or promotional images as **`banner.jpg`**.

*Note: Once you upload your images here, they will automatically be bundled and optimized for your Netlify and production deployment builds!*
